# Checklist de validation Homebridge

- Plugin dynamique.
- Package publié sur npm.
- Source GitHub avec issues activées.
- Releases GitHub avec notes.
- Support des versions Node.js LTS Homebridge actuelles.
- Configuration Homebridge UI.
- Pas d’accessoires inutiles si non configuré.
- Pas de télémétrie, tracking, analytics ou appels réseau inutiles.
- Pas de script post-install modifiant le système.
- Documentation claire.

Ce plugin est validé Homebridge. Cette checklist doit rester satisfaite pour les prochaines releases.
