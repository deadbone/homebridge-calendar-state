# Dates spéciales et exceptions

Les dates spéciales utilisent `MM-DD` et se répètent chaque année :

```json
{ "name": "Bastille Day", "date": "07-14" }
```

Les exceptions utilisent `YYYY-MM-DD` et ne concernent qu’une date :

```json
{
  "date": "2026-07-17",
  "dayOff": false,
  "workFromHome": true,
  "officeDay": false
}
```
