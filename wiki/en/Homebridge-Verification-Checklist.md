# Homebridge Verification Checklist

- Dynamic platform plugin.
- Published to npm.
- Source available on GitHub with issues enabled.
- GitHub releases for plugin versions.
- Supports current Homebridge Node.js LTS versions.
- Implements Homebridge Plugin Settings GUI.
- Does not start unnecessary accessories when unconfigured.
- Does not use telemetry, tracking, analytics, or unnecessary network calls.
- Does not run post-install system modification scripts.
- Provides clear user documentation.

This alpha is not yet Verified by Homebridge.
